# Wiring Reference

DIR Pins:
- GPIO 5  → Motor 1 Direction
- GPIO 14 → Motor 2 Direction
- GPIO 4  → Motor 3 Direction
- GPIO 22 → Motor 4 Direction

PWM Pins:
- GPIO 25 → Motor 1 PWM
- GPIO 26 → Motor 2 PWM
- GPIO 27 → Motor 3 PWM
- GPIO 23 → Motor 4 PWM

Note:
- Common ground required between ESP32 and motor driver
- Use separate motor power supply for stability

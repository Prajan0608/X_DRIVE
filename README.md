# ESP32 X-Drive Robot using PS5 Controller

This project implements an ESP32-based X-drive (omnidirectional) robot controlled via a PS5 Bluetooth controller.

## Features
- Omnidirectional movement (X-drive)
- PS5 joystick control
- PWM motor speed control
- Dead-zone filtering
- Scaled motor normalization

## Control Mapping
- Left Stick Y → Forward / Backward
- Left Stick X → Left / Right
- Right Stick X → Rotation

## Hardware
- ESP32 Dev Board
- 4 DC Motors with Omni Wheels
- Motor Driver (L298N / BTS7960)
- PS5 Controller

## Applications
- Robotics competitions
- AGV prototypes
- Academic projects
